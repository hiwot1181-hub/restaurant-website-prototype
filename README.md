# Restaurant Website - UI/UX Concept Prototype
## Overview
This project is a front-end website concept prototype for a modern restaurant.It focuses on structured layout, visual presentation, and user-friendly navigation.
## Features
- Home, Menu, Gallery, Contact sections
- Responsive layout
- Clean and Modern design
## Technologies Used
- HTML
- CSS
- JavaScript
## Purpose
Created to practice layout structuring, styling, user interface design concepts.
